Real-Time Filipino Sign Language Translator
Computer Vision Â· Accessibility Â· CNN Â· Python Â· OpenCV
Detailed
Project Overview

A real-time translator that recognizes Filipino Sign Language (FSL) hand gestures using a convolutional neural network (CNN) and computer vision, converting them into on-screen text. The system runs on a standard webcam setup.

Problem the System Solves

Communication between FSL users and non-signers is often limited by the lack of tools tailored specifically for FSL. This project offers a localized application that recognizes basic FSL gestures and displays them as text in real time.

Key Features
  • Real-time capture and processing of hand gestures with a webcam and OpenCV.
  • CNN-based classification of FSL gestures with approximately 88% accuracy.
  • On-screen translation of recognized gestures into text for easier interaction.
  • Optimized capture and inference pipeline that runs about 30% faster than the baseline implementation.
What I Learned
  • How dataset design impacts model performance and robustness.
  • Practical trade-offs between model accuracy and real-time responsiveness.
Development Process Summary

Methodology: Iterative development with short cycles for data collection, model training, evaluation, and interface refinement.

Tools Used: Python, OpenCV, TensorFlow/Keras, Jupyter Notebook, Git/GitHub.

Role & Contributions: End-to-end development, including dataset preparation, CNN architecture selection, training and tuning, performance evaluation, and integration of the trained model into the real-time application.
